# school-sim
Interactive school training simulator

I no longer have the credits to host the website using Google run. You must host the flask app yourself.

This project was made using Thonny for class with coding languages HTML, CSS, Jinja, Python, and a little bit of JavaScript.

1. Change your interpreter to this folder, it is a virtual environment.
2. Run app-test.py, this is the flask app.
3. The server should be running on http://127.0.0.1:8080

Gameplay

Read the tutorial then input the name of your student to start the game. 

Once you start you should see the following sections:

1. Display (Top Left) - Shows the date, the most recent event, and your character's name, hp, stats, and friends.
2. Event Log (Top Right) - A list of all events that your character has gone through.
3. Action Tab (Bottom Left) - Clickable buttons for actions to take, choose from the Training and Rest tab. 
4. Event Calendar (Bottom Right) - A calendar like quest log that shows the major events of your current year.
  
Keep choosing your actions until your character graduates or gets a bad end.

WARNING

The friends tab is a work in progress, buttons on that tab will redirect you to something that does not exist.
